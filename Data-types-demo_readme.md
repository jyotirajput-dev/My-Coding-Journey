# Data Types Demonstration in C

## Description
This program demonstrates:
- Basic data types in C (int, float, char)
- ASCII value of a character
- Arithmetic operations
- Use of printf() format specifiers

## Concepts Used
- int data type
- float data type
- char data type
- ASCII values
- Arithmetic operators
- printf() formatting

## Sample Output
The value of a is 5
The value of b is 9.50
The ASCII value of u is 117
The value of c is u
Sum of a and d is 83
The value of e is 145
