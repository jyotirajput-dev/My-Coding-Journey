# Sum of Two Numbers in C

## Description
This program takes two integer inputs from the user and prints their sum.

## Concepts Used
- int data type
- printf() function
- scanf() function
- %d format specifier
- Address-of operator (&)
- Basic arithmetic operation

## Sample Output
Enter the value of a: 5
Enter the value of b: 3
The sum of a and b is 8
