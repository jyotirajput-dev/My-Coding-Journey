# Area of Circle and Volume of Cylinder in C

## Description
This C program calculates:
- Area of a Circle
- Volume of a Cylinder

The program uses predefined values for radius and height.

## Formulas Used

Area of Circle = π × r × r

Volume of Cylinder = π × r × r × h

## Variables Used
- radius → Radius of the circle
- height → Height of the cylinder
- pi → Constant value of π (3.14)

## Sample Output

The area of the circle is 113.04
The volume of the cylinder is 339.12
