# Area of Circle and Volume of Cylinder (User Input) in C

## Description
This C program takes radius and height from the user and calculates:

- Area of a Circle
- Volume of a Cylinder

## Formulas Used

Area of Circle = π × r × r

Volume of Cylinder = π × r × r × h

## Concepts Used

- printf() for output
- scanf() for user input
- int and float data types
- arithmetic operations

## Sample Output

Enter the radius of the circle:
6

Enter the height of the cylinder:
3

The area of the circle is 113.04
The volume of the cylinder is 339.12
