# Area of Rectangle in C (User Input)

## Description
This C program takes the length and breadth of a rectangle from the user and calculates its area.

## Formula
Area = Length × Breadth

## Concepts Used
- printf() for output
- scanf() for input
- Integer variables
- Basic arithmetic operations

## Sample Output

Enter the length of the rectangle:
10

Enter the breadth of the rectangle:
5

The area of the rectangle is 50
