# Celsius to Fahrenheit Conversion in C

## Description
This C program converts a fixed temperature value from Celsius to Fahrenheit.

The program uses the formula:

F = (C × 9/5) + 32

Here the Celsius value is predefined as 37°C and the program calculates the equivalent Fahrenheit temperature.

## Concepts Used
- float data type
- printf() function
- arithmetic operations

## Sample Output

The value of this Celsius temperature in Fahrenheit is 98.60
